Follow these instructions to be able to run the project (Backend and Frontend).

Prerequisites:
1) Install the .NET 8 SDK.
2) Install Node.js (version 20.19.0 or higher) --> version 22.19.0 is recommended
3) Install Visual Studio Code and Visual Studio 2022 is recommended
4) Install Angular CLI: Install globally using the command
5) Clone the source code for the Backend and Frontend from this link: https://github.com/zaid4500/TaskManagementSystem.git


----------------------------------------------------------------------------------
Backend run steps:

After cloning the source code, open the Backend code in Visual Studio 2022; you'll find it in the folder "ManagementBE"

1) Set the project "ManagementBE.Presentation.API" as a startup project (if not already set)
2) Run the project



------------------------------------------------------------------------------------
Frontend run steps:

1) Open the Frontend code folder in Visual Studio Code; you'll find it in the folder "ManagmentUI"
2) Open the terminal and install npm (run this command "npm install" or run this command if any error returned "npm install --force")
3) When the npm is installed successfully, run the project (run this command "ng serve" or run this command if you're using a local version of node.js"npm start")
